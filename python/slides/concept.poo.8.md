##### Définition d'une classe 

```py
class Point(object):
    "Définition d'un point géométrique"
    pass

```

##### Objet : instance d'une classe 

```py
p9 = Point()

print(p9)

print(p9.__doc__)


```

##### Attributs (variables) d'instance

```py
>>> p9.x = 3.0

>>> p9.y = 4.0


```
