##### Les dictionnaires

[Les méthodes sur les dictionnaires](https://www.w3schools.com/python/python_ref_dictionary.asp)

* Déclaration d'un dictionnaire

```python
dico = {
    'computer': 'ordinateur', 
    'keyboard': 'clavier', 
    'mouse': 'souris'
}
```

* Méthodes spécifiques

```python
print(dico.keys())

print(dico.values())

print(dico.items())
```

* Taille dictionnaire

```python
len(dico)
```
